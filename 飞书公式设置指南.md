# 飞书表格公式设置指南

## 概述

由于飞书 API 不支持直接设置公式，计算后的数据将以数值形式写入飞书表格。如需在飞书表格中使用公式自动计算，请按照以下步骤手动设置公式。

## 公式设置步骤

### 1. 产品成本 (字段8)
**公式：** `产品成本RMB / 当前汇率`
- 使用除法公式
- 格式：数字（保留2位小数）

### 2. AMZ佣金 (字段9)
**公式：** `实时售价本币 * 0.15`
- 使用乘法公式（15%佣金）
- 格式：数字（保留2位小数）

### 3. 头程重量 (字段23)
**公式：** `包装重量_lb * 0.454`
- 将磅转换为公斤（1磅 = 0.454公斤）
- 格式：数字（保留2位小数）

### 4. 头程成本 (字段11)
**公式：** `(头程单价 / 当前汇率) * 头程重量`
- 格式：数字（保留2位小数）

### 5. 站内广告 (字段14)
**公式：** `实时售价本币 * 0.20`
- 使用乘法公式（20%广告费）
- 格式：数字（保留2位小数）

### 6. 退款费 (字段15)
**公式：** `实时售价本币 * 0.05`
- 使用乘法公式（5%退款费）
- 格式：数字（保留2位小数）

### 7. 含广利润 (字段17)
**公式：** `实时售价本币 - 产品成本 - AMZ佣金 - VAT - 头程成本 - FBA费 - FBA仓储费 - 站内广告 - 退款费 - 其他`
- 格式：数字（保留2位小数）

### 8. 含广利润率 (字段18)
**公式：** `含广利润 / 实时售价本币`
- 格式：百分比（保留2位小数）

### 9. 不含广利润 (字段19)
**公式：** `实时售价本币 - 产品成本 - AMZ佣金 - VAT - 头程成本 - FBA费 - FBA仓储费 - 退款费 - 其他`
- 格式：数字（保留2位小数）

### 10. 不含广利润率 (字段20)
**公式：** `不含广利润 / 实时售价本币`
- 格式：百分比（保留2位小数）

## 设置方法

### 在飞书表格中设置公式：

1. **点击字段标题**（如"产品成本"）
2. 选择"编辑字段"
3. 在"公式"选项中输入对应公式
4. 将字段类型设置为"数字"
5. 保存

### 批量设置提示：

- 公式字段中可以直接点击列名引用
- 例如：输入"实时售价本币"会自动引用该列数据
- 飞书会自动处理列引用，无需手动输入列号

## 注意事项

⚠️ **重要提示：**
1. **公式设置后，之前写入的数值会被自动替换为公式计算结果**
2. 建议先完成所有公式设置，再进行后续的数据分析
3. 如果需要保留原始计算值，可以复制表格到新的工作表

## 可编辑字段（无需公式）

以下字段需要手动输入或导入：
- 当前汇率
- VAT
- FBA仓储费
- 其他
- 产品成本RMB
- 头程单价

这些字段会在利润计算器中提供编辑功能，修改后会自动更新相关计算结果。
